Fetch Sequences & Publications from NCBI via Entrez

Help

This site allows you to fetch sequences or publications from the NCBI databases using a list of species you are interested in and a (nested) search term of your choice. So far you may query the Pubmed, Nucleotide & Protein databases.
eMail
Your email address is used to deliver you an email as soon as the results are ready. The email includes a link to download the found sequences (in FASTA-format) or publications (as CSV-file).
Search Term
Provide a search term (e.g. a gene name) you are interested in. You can nest your search term using the standard logical operations (AND/OR) and the field-IDs (e.g. CytB[Gene Name]) like produced by the different Search Builders of NCBI.
Species File
Upload a plain text file with all the species names you are interested in. Keep it to one species per line.
Database
Select the database you want to crawl. Right now you can choose between Pubmed, Nucleotide & Protein.
Download
You don't want to run it over the web? You can download the standalone Python-script, as well as the sources to this page. To run the commandline version you need to make sure to have installed Biopython.
